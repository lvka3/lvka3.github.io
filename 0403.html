防弟弟妹妹玩电脑 · 自动锁机脚本

开机后弹出密码窗口，60秒内输入正确密码 lvka 否则自动关机

⚙️ 工作原理：VBS脚本开机启动 → 倒计时60秒 → 输入正确密码“lvka”则取消关机，否则电脑自动关机。
✅ 无需安装任何软件，利用系统自带 shutdown 命令，适合Win7/Win10/Win11。
📜 Protect.vbs 完整代码 
' 防弟弟妹妹玩电脑脚本
' 开机后运行，要求在 1 分钟内输入密码 "lvka"，否则关机

Dim WshShell, startTime, elapsed, password, correct, timeoutSeconds, shutdownTimeout
Set WshShell = CreateObject("WScript.Shell")

' 先取消任何现有的关机计划，避免冲突
WshShell.Run "shutdown -a", 0, False

timeoutSeconds = 60          ' 用户输入时限（秒）
shutdownTimeout = timeoutSeconds + 5   ' 系统实际关机延迟（秒），预留缓冲

' 计划关机（65秒后执行）
WshShell.Run "shutdown -s -t " & shutdownTimeout, 0, False

startTime = Now()
correct = False

Do While Not correct
    elapsed = DateDiff("s", startTime, Now())
    If elapsed >= timeoutSeconds Then
        Exit Do        ' 超时，退出循环
    End If
    remaining = timeoutSeconds - elapsed
    password = InputBox("请在 " & remaining & " 秒内输入密码，否则电脑将自动关机。" & vbCrLf & vbCrLf & "密码：", "验证", "")
    
    If Not IsEmpty(password) Then
        If password = "lvka" Then
            correct = True
        Else
            If remaining > 0 Then
                MsgBox "密码错误！剩余 " & remaining & " 秒。", vbExclamation, "密码错误"
            End If
        End If
    Else
        ' 用户点击取消，继续计时，不提示错误
    End If
Loop

If correct Then
    ' 取消关机计划
    WshShell.Run "shutdown -a", 0, False
    MsgBox "密码正确，已取消关机。祝您使用愉快！", vbInformation, "验证成功"
Else
    ' 超时未输入正确密码，系统将自动关机
    MsgBox "未在限定时间内输入正确密码，电脑将关机。", vbCritical, "超时"
End If

📌 如何使用这个脚本（设置开机启动）

  1. 将上面代码复制到记事本中，另存为 Protect.vbs（注意扩展名必须是 .vbs，而不是 .txt）。
  2. 按下键盘 Win + R 打开运行窗口，输入 shell:startup 并回车，打开“启动”文件夹。
  3. 将刚才保存的 Protect.vbs 文件（或创建快捷方式）放入这个文件夹中。
  4. 重启电脑测试：开机进入桌面后会自动弹出密码输入框，60秒内输入 lvka 即可正常使用。
💡 小提示：如果弟弟妹妹强行结束进程（任务管理器里结束 wscript.exe），防护会失效。但对大部分孩子已经足够。
🔧 可以修改代码开头的 timeoutSeconds = 60 来调整倒计时秒数（例如改成 30）。
⚠️ 首次测试前请先保存好工作，以防意外关机。你也可以将关机延迟改成更大值方便调试。
📂 手动关闭/卸载方法：从“启动”文件夹中删除 Protect.vbs 即可。若脚本正在运行，可打开任务管理器结束 wscript.exe 进程。
